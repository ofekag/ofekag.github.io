GIS-Parking Search App

  • Tech Stack: JAVA, GOOGLE FIREBASE , GOOGLE MAP API, Android Studio software
  • Github URL: Project Link

GIS Application Development

The system was developed as an Android application (app), using Android Studio software. The geographic component we used: Google Maps API The database we used, Google's Firebase, in which we will save all the reports received in the system, which we will show the parking lots on the map.

• The user can report parking near him, by using the Google map, and providing relevant details regarding the parking (exact address, whether the parking is paid or not, whether the parking is covered or not, whether it is disabled parking or not).

• The user can search for parking near him, see the available parking on the map, filter according to his preferences, and like other users who reported parking.

• The user can view his personal profile, which includes an email address, first name, last name, mobile phone and username.

• The user can view the reports he made and the amount of likes he received.

The free parking spaces will be shown on the map, at the time the parking was reported, whether the parking is paid or not, whether the parking is covered or not, and whether it is handicapped parking or not.
